EasyTune 工具箱 使用说明
📌 软件简介
EasyTune 是一款轻量级、便携的硬件信息显示与软件启动管理工具。
它可以将你常用的各类检测工具、系统软件分类收纳，并提供简洁美观的毛玻璃界面，方便随时调用。

🔴🔴🔴请把需要的工具放在根文件夹的Tool文件夹里，再去程序中添加工具！！🔴🔴🔴

⚙️ 运行环境
操作系统：Windows 7 / 8 / 10 / 11（32位或64位均可）

运行库：无需额外安装，系统自带的 .NET Framework 4.7.2 即可运行
