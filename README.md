# EasyTune
EasyTune 是一款轻量级、便携的硬件信息显示与软件启动管理工具。
它可以将你常用的各类检测工具、系统软件分类收纳，并提供简洁美观的毛玻璃界面，方便随时调用。

本软件为个人兴趣开发作品，仅供学习交流使用。如在使用中遇到问题或有功能建议，欢迎联系开发者。
